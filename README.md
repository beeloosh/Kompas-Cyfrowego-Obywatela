# DCE Lesson Planner

DCE Lesson Planner to narzędzie edukacyjne bazujące na Planie Edukacji Obywatelstwa Cyfrowego Rady Europy. Ułatwia nauczycielom wyszukiwanie i planowanie efektów uczenia się oraz celów lekcji według wieku i obszarów tematycznych. To proste narzędzie internetowe umożliwiające tworzenie lekcji zgodnych z europejskimi standardami edukacji cyfrowej.

## Uruchomienie

1. Zainstaluj wymagania: `pip install streamlit pandas`
2. Uruchom: `streamlit run src/main.py`
3. Wybierz poziom wiekowy i domenę, aby przeglądać efekty uczenia się.
